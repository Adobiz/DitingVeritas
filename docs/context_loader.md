## 📡 闻境 — 语境预加载

  **实时翻译中自动注入领域术语，解决「一词多译」问题。**

### 痛点

  同声传译中，LLM 缺乏上下文容易直译错误：

| 无闻境                    | 有闻境                            |
| ---------------------- | ------------------------------ |
| "transformer" → 变压器    | "transformer" → Transformer 模型 |
| "attention" → 注意       | "attention" → 注意力机制            |
| "BERT is great" → 伯特很棒 | "BERT is great" → BERT 模型很优秀   |

### 方案

  用户输入 URL/关键词 → 智能抓取网页 → 自动提取术语 → 注入翻译提示词

1. **URL 抓取**：输入演讲主题链接，自动提取页面标题和正文

2. **关键词提取**：基于词频分析，过滤停用词，提取 Top 8 核心术语

3. **TTL 缓存**：60 分钟内重复使用同一语境，避免重复抓取

4. **热更新**：运行中可随时更新语境，下一句翻译立刻生效
   
   ### 效果
   
   ```yaml
   # 系统提示词（自动生成）
   你是实时同声传译 AI。将英文翻译为中文。
   
   当前语境/主题：Attention Is All You Need
   关键术语参考：attention, transformer, encoder, decoder, neural, network, BERT, GPT
   请在翻译时保持上述术语和语境一致。
   ```
- ✅ 术语翻译准确率显著提升
- ✅ 零额外 API 调用（纯规则提取）
- ✅ 支持 URL 和手动关键词两种输入
- ✅ 目前只支持有系统提示词（system prompt）功能的chat 模型使用该功能
